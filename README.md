This is a Next.js project with a complete autentication system. 

### Stack used 
- Next.js
- NextAuth
- Prisma ORM
- MongoDB

# Getting Started 

## Starting the dev server 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
